# Caprine - Unofficial Facebook Messenger App

**Description:** Unofficial Facebook Messenger app. This repository contains the core components and basic styling for a user-friendly, albeit unofficial, Messenger experience.

**Installation:**

1.  Clone the repository: `git clone https://github.com/your-username/caprine`
2.  Navigate to the project directory: `cd caprine`
3.  Install dependencies: `npm install` (or equivalent command for your package manager)

**Usage:**

*   **Basic Styling:** The `config.js` file contains basic styling directives.  The `browser.css` file provides the base layout for the browser.
*   **Core Components:** The `appveyor.yml` and `package.json` files outline the project's build process and dependencies, suitable for deployment to appveyor or similar platforms.
*   **JavaScript Files:**  The `index.js`, `menu.js`, `parse.js`, `dark-mode.css`, `media`, `build`, `src/`, and `static` directories contain the primary application logic and assets.
*   **Theme & UI:** `appveyor.yml` and `config.js` define the project's theme and user interface, including the use of `vibrancy.css` for a dark mode aesthetic.
*   **Maintain:**  This project is maintained by the user (you!).  Contributions are welcome, but please follow the project's guidelines and contribute responsibly.

**Files:**

*   `.editorconfig`: Configuration for code formatting.
*   `.gitattributes`: Git attributes for the repository.
*   `.gitignore`: Specifies files and directories to ignore.
*   `.travis.yml`:  Traverse script for automated testing and deployment.
*   `appveyor.yml`: Build configuration for appveyor deployment.
*   `browser.css`: Basic styling for the browser.
*   `browser.js`: JavaScript code for the browser.
*   `build`: Build script for the application.
*   `config.js`: Configuration file for the app.
*   `dark-mode.css`: Styles for a dark mode theme.
*   `index.js`: The main application entry point.
*   `license`: Project license information.
*   `media`:  Media assets for the app.
*   `menu.js`: The main menu functionality.
*   `package.json`: Project dependencies and metadata.
*   `readme.md`: Project documentation.
*   `static`: Static assets such as CSS and images.
*   `tray.js`: A tray component for visual feedback.
*   `vibrancy.css`: Styles for the vibrant user interface.
